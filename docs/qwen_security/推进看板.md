当前最早观察到的信号：

- `direct-only` 下：
  - `Stage A` 几乎完全可恢复
  - `Stage H` 几乎不可恢复
  - `Stage J / K` 处于中间
- `projection-enhanced` 下：
  - `Stage J / K` 的恢复率大幅上升，已逼近 `Stage A` baseline
  - `Stage H` 虽然上升，但仍明显低于部署线

这说明：

- 只看 `direct-only` 会低估部署线风险；
- 当前更强的 `VMA` 已足以揭示 `Stage J / K` 相对研究线的显著安全让步；
- `Gate 1` 已经满足阶段完成条件，下一步可进入 `Gate 2 / IMA`。

## 6. 已知约束

- 当前仓库尚无现成攻击脚本
- 当前 `outputs/security_qwen/` 已有首版 `VMA` 结果，但尚无 `IA / IMA / ISA / TFMA / SDA` 真实结果
- `Stage J / Stage K` 更偏部署线，不能默认继承 `Stage H` 的安全强度
- 中间态攻击只计入部署时真实可能暴露的中间量，不计入本机 hook / recorder
